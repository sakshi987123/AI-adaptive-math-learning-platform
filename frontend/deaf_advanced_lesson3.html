LESSON 3 · FAST PROBLEM SOLVING
Finding
Hypotenuse
Given the two legs a and b,
we calculate c using c = √(a² + b²)
c = √(a² + b²)
THE CORE FORMULA
a and b are legs (shorter sides).
c is hypotenuse — always the longest side, opposite the 90° angle.
STEP-BY-STEP · EXAMPLE 1
①
Write formula
c² = a² + b²
②
Substitute values
c² = 3² + 4² = 9 + 16
③
Add squares
c² = 25
④
Take square root
c = √25 = 5 ✓
EXAMPLE 2 · SPEED DRILL
①
Formula
c² = 5² + 12²
②
Square each
c² = 25 + 144
③
Add
c² = 169
④
Root
c = √169 = 13 ✓
