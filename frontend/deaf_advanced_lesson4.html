LESSON 4 · ADVANCED MISSING SIDE
Rearranging
the Formula
What if c is known but a leg is missing?
We rearrange a² + b² = c² to find a or b.
a² + b² = c²
THE THREE FORMS — REARRANGED
Find Hypotenuse (c)
c = √(a²+b²)
Use when: both legs known
Find Leg a
a = √(c²−b²)
Use when: c and b known
Find Leg b
b = √(c²−a²)
Use when: c and a known
Key insight: subtract the known square from c².
Always subtract — never add — when finding a missing leg.
EXAMPLE 1 · FIND LEG a
①
Start from Pythagoras
a² + b² = c²
②
Isolate a²
a² = c² − b²
③
Substitute c=5, b=4
a² = 25 − 16 = 9
④
Take square root
a = √9 = 3 ✓
EXAMPLE 2 · FIND LEG b (c=13, a=5)
①
Rearrange for b
b² = c² − a²
②
Substitute values
b² = 13² − 5² = 169 − 25
③
Subtract
b² = 144
④
Square root
b = √144 = 12 ✓
COMMON MISTAKES — AVOID THESE!
❌ WRONG ✅ CORRECT WHY?
a = √(a²+b²) − c a = √(c²−b²) Must subtract inside √
a² = c² + b² a² = c² − b² Leg uses minus, not plus
Forget √ at end Always take √ of result a²=9 → a=3, not 9
c is the shorter side c is always the longest Hypotenuse opposite 90°
LESSON 4 COMPLETE
🔍 Missing Side — Found!

a = √(c²−b²)

b = √(c²−a²)
Find leg b
Always subtract when finding a leg.
Always take the square root at the end!
