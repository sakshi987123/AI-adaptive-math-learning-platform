# AI Adaptive Learning System for Blind and Deaf Students

## Overview

The AI Adaptive Learning System is an accessibility-focused educational platform designed to make mathematics learning inclusive for visually impaired and hearing-impaired students. The project provides personalized learning experiences based on a student's learning ability and disability type, ensuring effective and engaging education for all learners.

This project focuses on teaching the **Pythagorean Theorem** through AI-powered adaptive learning methods.

## Problem Statement

Traditional learning systems often fail to address the accessibility needs of blind and deaf students. Blind learners struggle with visual mathematical content, while deaf learners may find audio-based explanations difficult to understand. Additionally, students learn at different speeds, but most educational platforms provide the same content to everyone.

## Proposed Solution

Our AI-powered adaptive learning platform personalizes educational content according to each student's learning level and accessibility requirements. The system uses Machine Learning, Generative AI, Explainable AI, and learning analytics to deliver customized learning experiences.

## Project Workflow

1. Student completes a pre-assessment test.
2. Machine Learning model classifies learning capability.
3. Personalized content is generated based on disability type and learning level.
4. Students learn the Pythagorean Theorem using adaptive content.
5. AI chatbot provides assistance and hints.
6. Post-assessment evaluates learning outcomes.
7. XAI-based feedback and recommendations are generated.
8. Progress is tracked through a personalized dashboard.
